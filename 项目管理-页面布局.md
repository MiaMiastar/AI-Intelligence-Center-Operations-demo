## 区域说明

| 区域 | 类名 / 元素 | 说明 |
|------|-------------|------|
| 整页容器 | `.admin` | Flex 布局：左侧边栏 + 右侧主内容 |
| 左侧边栏 | `.sidebar` | 固定宽度，深色背景；品牌 + 主导航（专家管理、课程管理、**项目管理**、权限设置） |
| 主内容区 | `.content` | 浅灰背景，内边距；包含头部 + 卡片区 |
| 页面头部 | `.content-header` | 标题「项目管理」+ 副标题「行业项目案例与可复制经验」 |
| 主卡片区 | `.card.card--projects` | 无边框、无阴影；内部：工具栏 + 状态区 + 网格 |
| 工具栏 | `.projects-toolbar` | 搜索框（占满剩余宽度）+ 结果数量 + 行业下拉 |
| 空状态 | `#projectState` | 无匹配结果时显示提示文案 |
| 项目网格 | `#projectGrid` | 自适应列数（minmax(320px, 1fr)），每格一个 `.project-card` |

## 单张项目卡片 (.project-card) 结构

```
┌─────────────────────────────────────────┐
│ [状态标签]                     [部署形态] │  ← .project-card__top
│ 项目名称                               │  ← .project-card__title
│ 🏢 所属单位                            │  ← .project-card__unit
│ [TRL7] [A] [商业化]                     │  ← .project-card__badges
│ 价值主张描述...                         │  ← .project-card__value
│ · 类型1 · 类型2 · 类型3                 │  ← .project-card__solutions
│ ┌─────────────────────────────────────┐ │
│ │ 落地案例    典型客户                 │ │  ← .project-card__stats
│ │ 3个        某整车厂 Tier1 供应商     │ │
│ │ 📈 关键指标                          │ │
│ └─────────────────────────────────────┘ │
│ 申请日期            [详情] [编辑]       │  ← .project-card__footer
└─────────────────────────────────────────┘
```

- 卡片无边框，仅有轻微阴影；悬停时阴影加深。
- 专家评审 / 待初审 的卡片右侧有彩色竖条（橙 / 蓝）。
